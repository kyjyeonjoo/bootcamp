# 목대 뭐먹지

목포대학교 주변 식당을 학생 관점에서 빠르게 찾아보는 간단한 웹서비스입니다.

## 사이트 주소

https://kyjyeonjoo.github.io/bootcamp/

## 만든 목적

목포대학교 학생이나 신입생이 공강 시간, 점심시간, 저녁 약속 전에 주변 식당 후보를 쉽게 고를 수 있도록 만들었습니다. 단순 맛집 소개가 아니라 가격대, 거리, 혼밥 가능 여부, 식사 상황을 기준으로 식당을 탐색하는 조사형 서비스입니다.

## 주요 기능

- 식당명, 메뉴, 상황 키워드 검색
- 한식, 분식, 중식, 카페, 고기 카테고리 필터
- 가격대 필터
- 혼밥 가능 식당만 보기
- 오늘의 식당 랜덤 추천
- 네이버 지도 검색 연결

## 참고

현재 식당 데이터는 웹서비스 예시 제작용 샘플입니다. 실제 제출 전에는 네이버 지도나 카카오맵을 기준으로 영업 여부, 메뉴 가격, 위치를 다시 확인해 업데이트하면 좋습니다.
