# 목대 뭐먹지

목포대학교 주변에서 식당을 고를 때 쓸 수 있는 간단한 조건 검색 웹서비스입니다.

## 사이트 주소

https://kyjyeonjoo.github.io/bootcamp/

## 만든 목적

목포대학교 학생들이 점심시간이나 공강 시간에 식당을 찾을 때 매번 지도 앱을 따로 검색해야 하는 불편함을 줄이기 위해 만들었습니다. 주변 식당 후보를 한 화면에 모아두고, 메뉴 종류와 가격대, 혼밥 가능 여부로 빠르게 골라볼 수 있게 구성했습니다.

또한 사용자의 식사 상황을 짧은 설문으로 입력받아 오늘의 식사 페르소나를 보여주고, 그 유형에 맞는 식당을 추천하도록 만들었습니다.

## 주요 기능

- 목포대 주변 식당 후보를 카드 형태로 확인
- 식당명, 메뉴, 상황 키워드 검색
- 한식, 분식, 중식, 카페, 고기, 해산물 등 종류별 필터
- 가격대별 필터와 혼밥 가능 필터
- 오늘의 식당 랜덤 추천
- 설문 조사 기반 목대생 식사 유형 분석
- 가성비 혼밥러, 카공 충전러, 든든밥파, 매운맛파, 친구모임파 등 페르소나 결과 제공
- 페르소나에 맞는 추천 식당 3개 표시
- 각 식당을 카카오맵 검색으로 바로 확인

## 페르소나 설계

설문은 함께 먹는 사람, 중요하게 보는 조건, 예산, 끌리는 메뉴를 기준으로 구성했습니다. 답변 결과를 바탕으로 사용자의 상황을 페르소나로 분류하고, 식당의 종류, 가격대, 혼밥 가능 여부, 메뉴 키워드를 점수화해 추천 식당을 정렬합니다.

## 조사 기준

식당 후보는 다이닝코드의 목포대 맛집 검색 결과와 지도 검색에 잡히는 상호명을 바탕으로 정리했습니다. 영업시간, 가격, 메뉴는 바뀔 수 있으므로 실제 방문 전에는 지도 앱에서 최신 정보를 다시 확인하는 것을 기준으로 했습니다.
